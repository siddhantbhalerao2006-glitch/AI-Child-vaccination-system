# 🧒 AI Child Vaccination System

An AI-powered backend system for managing child vaccination records, built using FastAPI, PostgreSQL, and SQLAlchemy. This project helps track immunization history, calculate upcoming vaccines based on India's National Immunization Schedule (NIS), and reduce missed vaccinations through intelligent insights.

---

## 🚀 Features

- 👶 Child Registration & Profile Management  
- 💉 Vaccination Record Tracking  
- 📜 Complete Vaccination History  
- 📅 Upcoming Vaccine Prediction (based on DOB + NIS)  
- ✏️ Update Vaccination Records  
- 🌐 REST API with FastAPI  
- ⚡ High-performance PostgreSQL database  
- 🔗 CORS enabled for React frontend integration  

---

## 🧠 AI Capabilities (Planned / Extendable)

- Predict missed vaccination risk  
- Smart reminders and alerts  
- District-wise vaccination analytics  
- Data-driven healthcare insights  

---

## 🛠️ Tech Stack

- **Backend:** FastAPI  
- **Database:** PostgreSQL  
- **ORM:** SQLAlchemy  
- **Validation:** Pydantic  
- **API Testing:** Swagger UI
